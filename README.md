# ふしぎノート

6歳の凪さんと一緒に、サイエンスゲートの教室のあと **好奇心を絶やさず、家でも理解を深め、子どもの「こういうのあったらいいね」を反映していく** 共創型のPWAです。

## できること

### 子どもが触る側
- **ふしぎボタン（？）**：ホーム右下のフローティング。タップで絵・声・写真・もじでアイデアを残せる
- **きょうのふしぎ**：開いた瞬間に1問だけ表示（親が前夜仕込んだ問い）
- **ふかぼりカード**：教室で習ったことを家で深ぼりする探究カード（観察・実験・収集・動画・対話）
- **質問のタネ**：「かあちゃんに きいてみよう」「たかちゃんに きいてみよう」など対話カード
- **6つのミッション**：絵を描く・写真・クイズ・おはなし（録音）・そうぞう・じっけん
- **音声読み上げ（🔊）**：問いや指示を音声で
- **「💌 かあちゃんとたかちゃんにみせる」**：作品を LINE/Gmail で共有
- **じぶんずかん**：完成したミッションを保存

### 親が触る側（おとな設定）
- **PINロック**：4桁PINで親メニューを保護
- **テーマ編集**：13ライブラリテーマ + 自由テーマを 個別に編集／コピー／削除
- **ふかぼりカード仕込み**：教室の内容を元に深ぼり問いを5〜7枚仕込む
- **質問のタネ仕込み**：家族との対話を誘うカードを書く
- **きょうのふしぎ仕込み**：日替わりで子供に出す問いをストック
- **アイデアばこ**：子供がふしぎボタンで残したものを見て、テーマ化／ふかぼり化
- **JSON入出力**：教科書写真をClaudeに渡してJSON生成→アプリにペースト
- **バックアップ**：ずかん全体を JSON ファイルで Google Drive 等に保存／復元

## 起動

```bash
cd ~/projects/fushigi-note
python3 -m http.server 4173
```

ブラウザで `http://127.0.0.1:4173` を開きます。

iPad/iPhone で試す場合は、同じ Wi-Fi 上で Mac のローカル IP を使って開き、Safari の共有メニューからホーム画面に追加します。

## 教科書からテーマを作る（30分→5分ワークフロー）

教室から帰ってきたら、教科書を写真にとって Claude に以下のプロンプトを送ります：

```
この教科書のページをもとに、6歳の子ども向けに「ふしぎノート」用のテーマJSONを作って。

# 出力形式（このフォーマット厳守）
{
  "icon": "🌱",
  "title": "テーマ名（30文字以内）",
  "question": "メインの問い（ひらがな多め・60文字以内）",
  "missions": [
    {"type": "draw", "title": "○○を えがこう", "prompt": "ひらがな多めの指示（120字以内）"},
    {"type": "photo", "title": "おうちで みつけよう",
     "prompt": "○○を さがして しゃしんを とろう",
     "hints": ["ヒント1", "ヒント2", "ヒント3"]},
    {"type": "quiz", "title": "クイズの題", "prompt": "問い",
     "choices": ["せんたくし1", "せんたくし2", "せんたくし3"], "answer": 0},
    {"type": "talk", "title": "○○について おはなししよう",
     "prompt": "おうちのひとに ○○の はなしを しよう",
     "hints": ["きっかけ1", "きっかけ2"]},
    {"type": "imagine", "title": "そうぞうしよう", "prompt": "もし ○○だったら？ えに かいてみよう"},
    {"type": "experiment", "title": "おうちで じっけん",
     "steps": ["ステップ1", "ステップ2", "ステップ3"]}
  ],
  "deepDives": [
    {"type": "observe",    "title": "○○を かんさつしよう", "prompt": "○○を よく みてみよう"},
    {"type": "experiment", "title": "おうちで じっけん",   "prompt": "○○を ○○してみよう"},
    {"type": "collect",    "title": "○○を あつめよう",    "prompt": "5つ さがして くらべよう"},
    {"type": "watch",      "title": "うごく ○○を みよう", "prompt": "YouTubeで みよう", "link": "https://youtube.com/..."},
    {"type": "talk",       "title": "○○について おしえて", "prompt": "なんで そう なるか かんがえよう"}
  ],
  "conversationSeeds": [
    "かあちゃんに、○○の はなしを きいてみよう",
    "たかちゃんに、○○について おしえてもらおう"
  ]
}

# 制約
- すべて ひらがな多め（漢字は最小限）
- 短く具体的に
- 全 6 ミッションを含める
- deepDives は 5〜7枚、conversationSeeds は 2〜4個
- 親（呼び方）は「かあちゃん」「たかちゃん」を使う（「ママ」「パパ」は使わない）
- JSON だけ返す（説明文不要）
```

返ってきたJSONを、ふしぎノートの **おとな設定 → 「📋 JSONで 作る・はりつける」** に貼り付け。

## おとな設定の使い方

1. ⚙ ボタンから設定画面へ
2. **「＋ あたらしい テーマを 作る」** で新規作成、または既存テーマの **「編集」** で上書き
3. テーマ名・アイコン・問いを入れる
4. ミッション 6種類それぞれを **トグルで ON/OFF**、ONなら内容を編集
5. **「💾 ほぞんして もどる」**
